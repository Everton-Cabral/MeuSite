    Projetos Desenvolvidos

    Formulário de Várias Etapas

    A minha aplicação é um formulário de várias etapas, que permite aos usuários realizar o cadastro e fazer escolhas das características de um serviço. Desenvolvida com a tecnologia Vue.js, a aplicação apresenta um layout responsivo, adaptando-se ao dispositivo utilizado. Foi utilizado o pré-processador SASS para a estilização dos componentes, seguindo a metodologia BEM, que traz uma padronização para a nomenclatura dos elementos e melhora a legibilidade do código, também foi utilizado o Vue Router para a navegação dos componentes.

    Fórum

    Desenvolvi uma aplicação de fórum onde os usuários podem realizar comentários, responder a outros comentários e curtir comentários. Todas as ações realizadas são armazenadas em um arquivo JSON, garantindo que as informações sejam salvas e acessíveis. O projeto foi uma ótima oportunidade para praticar minhas habilidades de programação, além disso, o projeto me permitiu trabalhar com arquivos JSON e gerenciamento de dados.

    Lista da Empregos

    O Sistema consome um arquivo JSON com informações sobre vagas de emprego e as exibe em uma lista organizada por categorias. O usuário pode filtrar a lista por categoria para encontrar facilmente as vagas de interesse. Além disso, eu utilizei Vuex como gerenciador de estado para garantir que a aplicação esteja sempre atualizada e sincronizada.

    Jogo da Velha

    O jogo inclui uma interface amigável e fácil de usar, permitindo que os jogadores façam suas jogadas com facilidade. Além disso, o jogo foi desenvolvido com responsividade, permitindo que ele se adapte facilmente a diferentes tamanhos de tela.

    Mega Sena

    O jogo é responsivo e projetado para funcionar em dispositivos móveis e de desktop.O jogo permite que os jogadores escolham seus números de aposta e salve suas jogadas no navegador do jogador, o que facilita a consulta de jogadas anteriores. O resultado da mega sena é chamado através de uma função random.
